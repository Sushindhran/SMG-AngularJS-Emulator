URL for the AngularJS Game Emulator
===================================

http://4-dot-smg-angularjs-emulator.appspot.com/


Directions on how to run the test.

1. Check out the code
2. Go to the base directory of the project and run the command

java -jar JsTestDriver-1.3.5.jar --port 4224

3. Now, Open up a browser window (chrome/ff) and hit the url http://localhost:4224. and capture the browser

4. open another terminal window, navigate to the base directory and run the command

java -jar JsTestDriver-1.3.5.jar --tests all
